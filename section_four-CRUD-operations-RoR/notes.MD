# Rails Naming Conventions

## For Example Article

## Model Name
> Singular, First letter uppercase(camelcase)

## Table Name
>Plural, lower case of model name

## Model name filename
> All lowercase but singular, article.rb

## Controller name
> Plural of model, so articles_controller.rb

# Another example

## Users

## Model name: User

## Model Filename
> user.rb

## Controller Name
